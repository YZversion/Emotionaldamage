上传微信聊天记录 JSON 文件

需已连接 API。后续评测会上传截断后的聊天内容到 OpenRouter。

支持的 JSON 格式示例 
// 格式一：带 is_send（推荐，可自动区分「我 / TA」）
[
  {
    "time": "2026-05-25 10:30:00",
    "is_send": true,
    "sender": "我",
    "content": "在干嘛呢"
  }
]

// 格式二：仅有 sender 时，导入后会请你选择「哪个是我」
{
  "chat": "联系人姓名",
  "messages": [
    {
      "timestamp": 1777610400,
      "sender": "Alice",
      "type": "text",
      "content": "想你了"
    }
  ]
}
👤

请选择：哪个是你？

文件里没有「是否本人发送」标记，需要你确认后才能正确做双向对比。
